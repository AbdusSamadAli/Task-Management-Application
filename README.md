# TaskFlow - Project Management System

## Description

TaskFlow is a role-based project management web application that allows teams to manage projects and tasks efficiently. The platform supports Admin and Member roles with secure authentication and real-time task tracking features.

Admins can create projects, assign tasks, manage workflows, and monitor progress, while Members can update assigned tasks and track deadlines.

The application provides a clean and responsive UI for managing team collaboration and productivity.

---

# Features

## Authentication & Authorization
- JWT-based Login and Signup
- Role-based access control
- Protected routes
- Admin and Member roles

## Project Management
- Create new projects
- Edit existing projects
- View project details
- Track project status
- Search projects

## Task Management
- Create and assign tasks
- Set priorities and due dates
- Update task status
- Track task progress
- Search tasks

## Dashboard
- Overview of projects and tasks
- User-specific task visibility
- Real-time workflow management

## Responsive UI
- Modern responsive interface
- Sidebar navigation
- Modal-based forms
- Clean dashboard layout
